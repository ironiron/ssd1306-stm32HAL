◆ HardwareConf

enum SSD1306::HardwareConf : uint8_t

This enum is for setting hardware configuration. Devices from different vendors can have OLED segments internally mapped to different SSD1306 pins. Try ALT_NOREMAP for 128x64, and SEQ_NOREMAP for 128x32 as a rule of thumb.
