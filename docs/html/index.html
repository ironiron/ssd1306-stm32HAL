SSD1306 Documentation

Library for controlling OLED displays based on SSD1306 drivers.

It is based on library made by Olivier Van den Eede. One of the main modifications include separation of hardware related functions. Additionally there are some extra functions not present in original work It is developed with stm32 and I2C in mind, but can be easily modified for other microcontrollers, by modifying SSD1306_hardware.cpp and SSD1306_hardware_conf.hpp files only
